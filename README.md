SISTEMA DE ESTOQUE
----

Projeto desenvolvido em linguagem C com o objetivo de praticar:
Estruturas de decisão (if, else, switch)
Estruturas de repetição (do while)
Matrizes e estruturas de repetição aninhadas
Lógica de programação
----

🎮 Sobre o Projeto
Este é um Sistema de estoque com:

✅ Menu interativo
✅ Cadastro de produtos
✅ Adicao ou subtracao de quantidade dos itens
✅ Busca interativa dos produtos
✅ Controle de estoque anti negativo

---

Modularizacao incrementada para facilitar manutencao e leitura.
---

🛠 Tecnologias Utilizadas

Linguagem C
Compilador GCC
Terminal / Console
